About the project

The purpose of this project is to investigate the concept of an interactive virtual art gallery. It will be created with the context of connecting the gallery spaces at the University of Pennsylvania. There are a collection of galleries scattered around campus, however they are both separated by distance and also type of artwork. I want to build a website that allows users to navigate between and through the galleries. Investigating the optimal User Interface and User Experience to help connect the galleries and artwork.  I hope to collect data from the users as they progress through the galleries and use the data to influence the gallery experience.
